## How to test locally on Windows
```powershell
cd orchestrator
npm install
npm test
cd ../desktop-runner
python -m pytest
```

## Manual verification steps (Notepad)
1. Open Notepad.
2. From the repo root, run:
   ```powershell
   python desktop-runner/scripts/manual_notepad_test.py
   ```
3. Confirm the script pastes text into Notepad and reports success.
